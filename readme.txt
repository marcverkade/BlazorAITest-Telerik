BlazorAITest

Sample application using TelerikGrid and extend with an extra property and add keyboard navigation.
The name expects AI test but that ius pending, first some Telerik and other handy stuff

********
Features
********
* Use of Telerik components
* Use of Toastmessages
* Automatic focussing first control after loading pages
* Use of ILogger to see messages in the console
* Extende TelerikGrid with keyboard navigation
  - [ARROWUP] and [ARROWDOWN] key navigation
  - [ENTER] key for editing selected row
  - Automatic focus on the first Textbox so you can start filtering instantly

***************
Technology used
***************
- Visual Studio 2022
- .Net Core 9
- Blazor Telerik UI
- Javascript
